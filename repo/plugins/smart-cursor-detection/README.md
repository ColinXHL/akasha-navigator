# 智能鼠标检测插件

检测游戏内鼠标状态，自动切换播放器点击穿透模式。

## 功能

- 监控指定游戏进程的鼠标状态
- 当游戏显示鼠标（UI 模式）时，自动启用点击穿透并降低透明度
- 当游戏隐藏鼠标（游戏模式）时，自动禁用点击穿透并恢复透明度
- 自动点击穿透与手动点击穿透独立，互不影响

## 配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| processWhitelist | 监控进程列表（逗号分隔） | 空 |
| minOpacity | UI 模式最低透明度 | 0.3 |
| intervalMs | 检测间隔（毫秒） | 200 |

## 使用方法

1. 安装插件
2. 在插件设置中配置监控进程列表，例如：`YuanShen, GenshinImpact`
3. 启动游戏，插件会自动检测鼠标状态并切换穿透模式

## 支持的游戏

- 原神（YuanShen, GenshinImpact）
- 其他支持鼠标隐藏的游戏

## 权限

- `window` - 控制窗口透明度和点击穿透
- `events` - 监听鼠标状态事件
